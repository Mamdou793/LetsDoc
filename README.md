# LetsDoc — Local AI Document Assistant (MVP)

LetsDoc is a cross‑platform Avalonia desktop application that loads documents and answers questions using local AI.

## Features (MVP)
- Clean Avalonia UI (3‑panel layout)
- Query panel with reactive commands
- Document viewer
- Document list sidebar
- Working MVVM architecture
- Thread‑safe async operations

## Roadmap
- File picker (macOS)
- PDF/TXT/Markdown loading
- Embeddings + vector search
- Local LLM inference
- `.app` packaging for macOS
